# Code of Conduct

## Our Pledge

We want this project to be a respectful, practical place to improve crash-report analysis tooling. Contributors are expected to be considerate, direct, and focused on the work.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect differing experience levels and constraints.
- Give constructive technical feedback.
- Avoid sharing private crash reports, proprietary symbols, or user-identifying data in public issues or pull requests.

## Unacceptable Behavior

- Harassment, insults, threats, or sustained disruptive behavior.
- Publishing someone else's private information.
- Posting sensitive crash artifacts without permission.

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate this code of conduct. Repeated or severe violations may result in a ban from project spaces.
