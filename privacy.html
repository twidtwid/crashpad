CrashPad

Privacy Policy

CrashPad is designed to analyze Apple crash reports in your browser. Uploaded crash reports are not sent to this server and are not stored by the application.

What The App Processes

When you choose or drop a crash report, the file is read by browser JavaScript in the current tab so the report can be parsed and rendered. The app keeps the parsed report in page memory only.

What The Server Stores

The server hosts the static app and one public QLThumbnail example report. It does not provide an upload endpoint, write uploaded reports to disk, or persist uploaded report contents.

Ephemeral Controls

Click Forget Report, reload the page, or close the tab to clear the current report from browser memory. Export JSON and Copy Summary are local browser actions you initiate.

Operational Logs

Hosting infrastructure may record ordinary request metadata such as IP address, URL path, user agent, and timestamps. Do not put sensitive crash data in URLs. Crash report file contents are not transmitted by the upload control.

Public Example

The built-in QLThumbnail example is a sanitized fixture intended for demonstration and testing.
